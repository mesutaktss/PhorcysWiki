Metin Üretici

Genel Bakış

`Metin Üretici` sınıfı, Ollama API kullanarak metin üretme yöntemlerini sağlar. Belirli bir modelden yanıtlar üretme, dummy metin oluşturma ve yanıt üretim sürecinde meydana gelen hataları yönetme gibi özellikleri destekler.

Özellikler

  • Metin Üretimi: Verilen bir isteme dayanarak metin üretir ve Ollama API kullanır.
  • Yanıt Yönetimi: Ollama sohbet hizmetinden gelen yanıtları ve hataları yönetir.
  • Dummy Metin Oluşturma: Belirli bir kelime sayısına sahip dummy metin oluşturur.

Kullanım

Üreticiyi Başlat

generator = TextGenerator(
    model_name="brooqs/mistral-turkish-v2"
)

Belirtilen model adı ile `TextGenerator` sınıfının bir örneğini oluşturur.

Metin Üret

generator.generateText(
    prompt
)

Belirtilen model kullanılarak sağlanan isteme dayanarak metin üretir.

Yanıt Üret

TextGenerator.generateResponse(
    model_name, 
    prompt
)

Belirtilen model ve istemden yanıt üretir. `ollama.chat` işlevi başarısız olursa hataları yönetir.

Dummy Metin Oluştur

TextGenerator.createDummyText(
    num_words
)

Önceden tanımlanmış bir dosyadan belirli sayıda kelime içeren dummy metin oluşturur.
